At Madisonium we like to work on sisyphean goals by defining discrete, manageable projects.

If you’d like to help the web and larger tech industry in Madison, please join a project team and pitch in, or start a new project that we can help funnel resources into.

Current Projects

  1. Getting Started Guide – A guide to the Madison tech scene.
  2. Made of Madison – A brand to represent apps/products made in Madison.
  3. Web District – An initiative to develop the East Rail Cooridor into a home for the web industry in Madison.
